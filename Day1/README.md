# LangChain Learning Journey 🚀

This repository contains my progress as I learn to build Generative AI applications using LangChain, Ollama, and OpenAI.

## 🛠️ Projects Included
- **Marketing Email Generator (`assignment4.py`)**: A sequential chain that generates catchy subject lines and structured JSON emails.
- **Speech Architect (`format_output.py`)**: A multi-model pipeline (Llama 3.2 + GPT-4o) for creating emotional speeches.
- **LCEL Demos**: Experiments with LangChain Expression Language and custom parsers.

## 🧰 Tech Stack
- **Framework**: LangChain
- **LLMs**: Ollama (Llama 3.2, Mistral), OpenAI (GPT-4o)
- **UI**: Streamlit
- **Environment**: Python 3.11+
